⚙️  CONFIGURAÇÃO NECESSÁRIA:

1. Backend/.env (criar a partir de .env.example)
   ✓ DB_HOST, DB_USER, DB_PASSWORD, DB_DATABASE
   ✓ JWT_SECRET (gerar chave aleatória 32+ chars)
   ✓ JWT_REFRESH_SECRET (diferente)
   ✓ EMAIL_USER, EMAIL_PASSWORD (opcional)
   ✓ CORS_ORIGIN (seu domínio)

2. MySQL Database
   ✓ Importar database/carteira_database.sql

3. Dependências Node
   ✓ npm install no diretório /backend

4. Iniciar Servidor
   ✓ npm start (produção)
   ✓ npm run dev (desenvolvimento com nodemon)
